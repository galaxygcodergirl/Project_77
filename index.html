Project on Console

Visual Studio Code is where you can create a web page. To set it up you have to create of folder on desktop and upload it in to visual studio code. After you create index.html, style.css and maybe script.js you are free to code your own web page about anything.

The console screen is all about writing something, you can use it to show square roots, your full name in concat, adding and subtracting numbers. To open console you right-click on the page and there will be a dropdown. Select inspect and the click on console on your new screen. To write anything on the console screen you can go to script.js if you have alot of code to do. But, if you have very less code open up the tag script and start writing your code down below.
